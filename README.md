# Slide Manager Dashboard

A collaborative PowerPoint/Google Slides management tool built with Streamlit that allows team members to upload, view, and edit slides in a unified dashboard.

## Features

### 1. **User Authentication**
- Login/Logout system
- Role-based access (Admin/Member)
- Session management

### 2. **Slide Management**
- Upload PowerPoint/PDF files
- Add metadata (title, description, tags)
- Categorize slides
- Search and filter functionality

### 3. **Collaboration Features**
- Comment on slides
- Edit permissions control
- Share slides with team members

### 4. **Dashboard**
- Overview metrics
- Grid/list view of all slides
- File preview and download

### 5. **Team Management** (Admin only)
- Add/remove team members
- Reset passwords
- View activity logs
